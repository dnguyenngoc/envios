# ENVIOS

This is project for handle restore Iphone.

## How to run

### 1. Build libidevice lib/ python requirements/ nodejs dependency
```
  sh build_debian.sh
```

### 2. Run

```
  sh run_debian.sh
```

### 3. Stop

```
  sh stop.sh
```
